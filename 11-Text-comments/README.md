# Проект: Классифицикация текстов комментариев на позитивные и негативные

***Выбран проект без BERT, чтобы научиться работать с тяжелыми проектами в условиях ограниченных ресурсов, без GPU***

<h2>Описание проекта</h2>
Интернет-магазин «Викишоп» запускает новый сервис. Теперь пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Магазину нужен инструмент, который будет искать токсичные комментарии и отправлять их на модерацию. 
<br>Обучите модель классифицировать комментарии на позитивные и негативные. В вашем распоряжении набор данных с разметкой о токсичности правок.
<br>Постройте модель со значением метрики качества <b>F1 не меньше 0.75.</b>
<h2>Описание данных</h2>
Данные находятся в файле <b>/datasets/toxic_comments.csv</b>
Столбец <b>text</b> в нём содержит текст комментария, а <b>toxic</b> — целевой признак.
