# Проект: прогнозирование температуры нагрева сплава

<h2>Описание проекта: промышленность</h2>
Чтобы оптимизировать производственные расходы, металлургический комбинат «Стальная птица» решил уменьшить потребление электроэнергии на этапе обработки стали. Для этого комбинату нужно контролировать температуру сплава. Ваша задача — построить модель, которая будет её предсказывать. 
<br>Заказчик хочет использовать разработанную модель для имитации технологического процесса. Ниже расскажем о деталях этого процесса. Их важно знать, прежде чем генерировать новые признаки.
<h3>Описание этапа обработки</h3>
<ul>
<li>Сталь обрабатывают в металлическом ковше вместимостью около 100 тонн. Чтобы ковш выдерживал высокие температуры, изнутри его облицовывают огнеупорным кирпичом. Расплавленную сталь заливают в ковш и подогревают до нужной температуры графитовыми электродами. Они установлены в крышке ковша.</li>
<li>Из сплава выводится сера (этот процесс — десульфурация), добавлением примесей корректируется химический состав и отбираются пробы. Сталь легируют — изменяют её состав — подавая куски сплава из бункера для сыпучих материалов или проволоку через специальный трайб-аппарат (от англ. tribe — «масса»).</li>
<li>Перед тем как первый раз ввести легирующие добавки, измеряют температуру стали и производят её химический анализ. Потом температуру на несколько минут повышают, добавляют легирующие материалы и продувают сплав инертным газом. Затем его перемешивают и снова проводят измерения. Такой цикл повторяется до достижения целевого химического состава и оптимальной температуры плавки.</li>
<li>Тогда расплавленная сталь отправляется на доводку металла или поступает в машину непрерывной разливки. Оттуда готовый продукт выходит в виде заготовок-слябов (от англ. slab — «плита»).</li>
</ul>
<h2>Описание данных</h2>
Данные состоят из нескольких файлов, полученных из разных источников:
<ul>
<li><b>data_arc_new.csv</b> — данные об электродах;</li>
<li><b>data_bulk_new.csv</b> — данные о подаче сыпучих материалов (объём);</li>
<li><b>data_bulk_time_new.csv</b> — данные о подаче сыпучих материалов (время);</li>
<li><b>data_gas_new.csv</b> — данные о продувке сплава газом;</li>
<li><b>data_temp_new.csv</b> — результаты измерения температуры;</li>
<li><b>data_wire_new.csv</b> — данные о проволочных материалах (объём);</li>
<li><b>data_wire_time_new.csv</b> — данные о проволочных материалах (время).</li>
</ul>

<h3>Файл data_arc_new.csv</h3>
<ul>
<li><b>key</b> — номер партии;</li>
<li><b>Начало нагрева дугой</b> — время начала нагрева;</li>
<li><b>Конец нагрева дугой</b> — время окончания нагрева;</li>
<li><b>Активная мощность</b> — значение активной мощности;</li>
<li><b>Реактивная мощность</b> — значение реактивной мощности.</li>
</ul>

<h3>Файл data_bulk_new.csv</h3>
<ul>

</ul>

<h3>Файл data_bulk_time_new.csv</h3>
<ul>
<li><b>key</b> — номер партии;</li>
<li><b>Bulk 1 … Bulk 15</b> — время подачи материала.</li>
</ul>

<h3>Файл data_gas_new.csv</h3>
<ul>
<li><b>key</b> — номер партии;</li>
<li><b>Газ 1</b> — объём подаваемого газа.</li>
</ul>

<h3>Файл data_temp_new.csv</h3>
<ul>
<li><b>key</b> — номер партии;
<li><b>Время замера</b> — время замера;</li>

</ul>

<ul>

<li><b>Wire 1 … Wire 9</b> — объём подаваемых проволочных материалов.</li>
</ul>

<h3>Файл data_wire_time_new.csv</h3>
<ul>
<li><b>key</b> — номер партии;</li>

</ul>
